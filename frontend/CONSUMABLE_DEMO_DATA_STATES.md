# 耗材演示数据状态说明

## 数据多样性概述

更新后的耗材演示数据覆盖了所有可能的库存量和有效期状态组合，能够完整演示耗材管理系统的各种逻辑。

## 库存量分类标准
- **库存充足**：当前库存 > 50% 最大库存
- **库存不足**：当前库存 ≤ 50% 最大库存且 > 0
- **无库存**：当前库存 = 0

## 有效期分类标准
- **已过期**：有效期已过当前日期
- **即将过期**：有效期在30天内
- **正常有效期**：有效期在30天以上
- **长期有效**：有效期在2年以上
